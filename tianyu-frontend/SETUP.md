# TIANyu Frontend - Setup Guide

## Prerequisites
- Node.js 18+ 
- npm or yarn or pnpm

## Installation
```bash
cd tianyu-frontend
npm install
# or
yarn install
# or
pnpm install
```

## Development
```bash
npm run dev
# Opens on http://localhost:3000
```

## Production Build
```bash
npm run build
npm start
```

## Linting
```bash
npm run lint
```

## Formatting
```bash
npx prettier --write .
```

## Available Components
- Button
- Input
- Tab
- Card
- Badge
- Layout (Navbar + Footer)

## Theming
Edit `tailwind.config.js` to customize colors.
