## 1. 色彩系统

### 1.1 品牌色（WEB UI 新版）

| 名称 | 色值 | 渐变 | 用途 |
|------|------|------|------|
| **琥珀橙** | `#E67E22` | - | 主按钮、Tab 激活态、标题强调、链接、CTA |
| **深橙** | `#D35400` | - | 按钮渐变终点、Hover 态、点击态 |
| **品牌渐变** | - | `#E67E22 → #D35400` | 主按钮背景、重要 CTA |

### 1.2 辅助色

| 名称 | 色值 | 用途 |
|------|------|------|
| **奶油米** | `#FAF3E0` | 背景色、卡片装饰、Banner 背景、分隔区域 |
| **暖米色** | `#F5E6D3` | 次级背景、卡片悬停态、选中态背景 |
| **浅金** | `#F9E7C4` | 高亮背景、提示框背景 |

### 1.3 中性色

| 名称 | 色值 | 用途 |
|------|------|------|
| **标题黑** | `#1A1A1A` | 页面标题、重要文字 |
| **正文黑** | `#333333` | 模块标题、正文内容 |
| **正文灰** | `#666666` | 次要内容、说明文字 |
| **辅助灰** | `#999999` | 占位符、次要信息、未激活态 |
| **冷灰** | `#F0F0F0` | 背景、分割线、边框、未激活 Tab |
| **边框灰** | `#E0E0E0` | 输入框边框、卡片边框 |
| **卡片白** | `#FFFFFF` | 卡片背景、页面背景 |

### 1.4 功能色

| 名称 | 色值 | 用途 |
|------|------|------|
| **红色标签** | `#FF4D4F` | "老板必用"标签背景、错误提示、警告 |
| **绿色标签** | `#52C41A` | 绿色标签系列、成功状态、完成提示 |
| **蓝色标签** | `#3498DB` | 蓝色标签系列、信息提示、帮助链接 |
| **紫色标签** | `#9B59B6` | 特殊标签、VIP 标识 |
| **黄色标签** | `#F1C40F` | "上新"标签、注意提示 |

### 1.5 渐变规范

```css
/* 主按钮渐变 - 琥珀橙 */
.primary-gradient {
  background: linear-gradient(135deg, #E67E22 0%, #D35400 100%);
}

/* 标签渐变（老板必用） */
.premium-gradient {
  background: linear-gradient(135deg, #FF6B35 0%, #FF4D4F 100%);
}

/* 页面背景渐变 - 奶油米 */
.page-bg-gradient {
  background: linear-gradient(180deg, #FAF3E0 0%, #FFFFFF 100%);
}

/* 卡片背景渐变 */
.card-bg-gradient {
  background: linear-gradient(135deg, #FFFFFF 0%, #FAF3E0 100%);
}

/* 琥珀金渐变（特殊活动） */
.amber-gold-gradient {
  background: linear-gradient(135deg, #F1C40F 0%, #E67E22 100%);
}
```

### 1.6 色彩使用原则

| 场景 | 推荐用色 |
|------|----------|
| 主要操作按钮 | 琥珀橙渐变 `#E67E22 → #D35400` |
| 次要操作按钮 | 白色 + 琥珀橙描边 `#E67E22` |
| 激活状态 | 琥珀橙 `#E67E22` |
| 未激活状态 | 辅助灰 `#999999` |
| 页面背景 | 奶油米 `#FAF3E0` 或 冷灰 `#F0F0F0` |
| 卡片背景 | 白色 `#FFFFFF` + 奶油米装饰 |
| 重要标签 | 红色/黄色/琥珀橙 |
| 普通标签 | 蓝色/绿色/紫色 |
| 成功提示 | 绿色 `#52C41A` |
| 错误提示 | 红色 `#FF4D4F` |
| 警告提示 | 黄色 `#F1C40F` |
| 信息提示 | 蓝色 `#3498DB` |

### 1.7 WEB UI 配色方案

#### 整体风格
- **主色调**: 琥珀橙 `#E67E22` - 温暖、活力、专业
- **辅助色**: 奶油米 `#FAF3E0` - 柔和、高级、舒适
- **中性色**: 冷灰 `#F0F0F0` - 现代、简洁、百搭

#### 情绪板
```
温暖活力 × 高级轻奢 × 现代简洁

琥珀橙 → 热情、行动力、创造力
奶油米 → 温馨、舒适、品质感
冷灰   → 专业、冷静、平衡
```

## 2. 字体排印

### 2.1 字号体系

| 层级 | 字号 | 字重 | 行高 | 字间距 | 用途 |
|------|------|------|------|--------|------|
| **超大标题** | 28px | Bold (700) | 1.4 | 0 | Banner 标题、页面大标题 |
| **大标题** | 24px | Bold (700) | 1.4 | 0 | 页面标题 |
| **小标题** | 20px | SemiBold (600) | 1.4 | 0 | 模块标题 |
| **副标题** | 18px | SemiBold (600) | 1.4 | 0 | 卡片标题 |
| **正文大** | 16px | Regular (400) | 1.6 | 0 | 主要内容、按钮文字 |
| **正文** | 14px | Regular (400) | 1.5 | 0 | 说明文字、辅助信息 |
| **小字** | 12px | Regular (400) | 1.4 | 0 | 标签、提示、消耗说明 |

### 2.2 字体颜色

| 层级 | 色值 | 用途 |
|------|------|------|
| 主标题 | `#1A1A1A` | 页面标题、重要强调 |
| 次标题 | `#333333` | 模块标题、卡片标题 |
| 正文 | `#666666` | 主要内容、表单标签 |
| 辅助 | `#999999` | 占位符、次要信息、提示 |
| 链接 | `#1E80FF` | 可点击链接、帮助文字 |

### 2.3 字体栈

```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, 
             "Helvetica Neue", Arial, "Noto Sans SC", "PingFang SC", 
             "Microsoft YaHei", sans-serif;
```

### 2.4 字体使用原则

| 场景 | 推荐样式 |
|------|----------|
| 页面标题 | 24px Bold, #1A1A1A |
| 模块标题 | 18-20px SemiBold, #333333 |
| 表单标签 | 16px Bold, #333333 |
| 输入框文字 | 16px Regular, #333333 |
| 占位符 | 14px Regular, #999999 |
| 按钮文字 | 16px Bold, 白色 |
| 标签文字 | 12px Regular, 白色 |
| 消耗提示 | 12px Regular, #999999 |

## 3. 间距系统

### 3.1 基础单位

**基准值：4px**

所有间距应为 4px 的倍数：4, 8, 12, 16, 20, 24, 32, 40, 48...

### 3.2 页面间距

| 名称 | 数值 | 用途 |
|------|------|------|
| 页面边距 | 16px | 页面左右安全边距 |
| 页面上下边距 | 24px | 页面顶部/底部边距 |
| 模块间距 | 24px | 大模块之间的垂直间距 |

### 3.3 组件间距

| 名称 | 数值 | 用途 |
|------|------|------|
| 卡片间距 | 12px | 卡片之间的间距 |
| 卡片内边距 | 16px | 卡片内容与边框的间距 |
| 元素间距 | 8px | 图标与文字、标签与内容 |
| 字段间距 | 20px | 表单字段之间的垂直间距 |
| 按钮间距 | 12px | 并排按钮之间的间距 |

### 3.4 输入框间距

| 名称 | 数值 | 用途 |
|------|------|------|
| 输入框高度（单行） | 48px | 单行文本输入框 |
| 输入框高度（多行） | 80px | 多行文本输入框 |
| 输入框内边距 | 12px 16px | 上下 12px，左右 16px |
| 标签与输入框间距 | 8px | 字段标签与输入框的间距 |

### 3.5 CSS 变量

```css
:root {
  /* 页面间距 */
  --spacing-page-margin: 16px;
  --spacing-page-vertical: 24px;
  --spacing-module: 24px;
  
  /* 组件间距 */
  --spacing-card-gap: 12px;
  --spacing-card-padding: 16px;
  --spacing-element: 8px;
  --spacing-field: 20px;
  --spacing-button-gap: 12px;
  
  /* 输入框 */
  --input-height-sm: 40px;
  --input-height-md: 48px;
  --input-height-lg: 80px;
  --input-padding: 12px 16px;
}
```

## 4. 布局系统

### 4.1 页面框架

```
┌─────────────────────────────────────┐
│   状态栏 (24px)                     │
├─────────────────────────────────────┤
│   导航栏 (44px)                     │
│   [返回] 页面标题 [...] [更多]       │
├─────────────────────────────────────┤
│   Banner 区 (可选，120-200px)       │
├─────────────────────────────────────┤
│   Tab 切换区 (可选，44px)           │
├─────────────────────────────────────┤
│   说明文字区 (可选，auto)           │
├─────────────────────────────────────┤
│   内容区 (自适应，滚动)             │
│   - 表单字段                        │
│   - 卡片列表                        │
│   - 网格布局                        │
├─────────────────────────────────────┤
│   底部操作区 (固定，80px)           │
│   [次按钮]          [主按钮]        │
├─────────────────────────────────────┤
│   TabBar (50px + 安全区)            │
└─────────────────────────────────────┘
```

### 4.2 栅格系统

#### 小程序端（固定宽度 375px）

| 网格类型 | 列数 | 列宽 | 间距 |
|----------|------|------|------|
| 功能网格（大卡） | 2 列 | 167px | 12px |
| 功能网格（中卡） | 4 列 | 78px | 12px |
| 功能网格（小卡） | 5 列 | 59px | 8px |
| AI 工具卡片 | 2 列 | 167px | 12px |

#### WEB 端（响应式）

| 断点 | 列数 | 列宽 | 最大宽度 |
|------|------|------|----------|
| Mobile (<768px) | 2 列 | 自适应 | 100% |
| Tablet (768-1024px) | 3 列 | 自适应 | 100% |
| Desktop (>1024px) | 4 列 | 280px | 1200px |

### 4.3 圆角规范

| 元素 | 圆角 | 用途 |
|------|------|------|
| 按钮 | 24px | 主按钮、次按钮 |
| 输入框 | 8px | 文本输入框、下拉选择 |
| 卡片 | 12-16px | 功能卡片、AI 工具卡片 |
| Tab | 20px | Tab 切换按钮 |
| 标签 | 4px | Badge 标签 |
| 弹窗 | 16px | Modal、Dialog |

### 4.4 阴影规范

```css
/* 卡片阴影 */
.card-shadow {
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.04);
}

/* 悬浮阴影 */
.hover-shadow {
  box-shadow: 0 4px 16px rgba(0, 0, 0, 0.08);
}

/* 按钮阴影 */
.button-shadow {
  box-shadow: 0 2px 12px rgba(30, 128, 255, 0.2);
}
```

```css
/* 在组件中使用变量 */
.my-component {
```
