## 1. 开发环境搭建

### 1.1 系统要求

| 软件 | 最低版本 | 推荐版本 |
|------|----------|----------|
| Node.js | v16.x | v20.x LTS |
| npm | v8.x | v10.x |
| Git | v2.x | v2.40+ |
| 浏览器 | Chrome 90+ | Chrome 最新 |

### 1.2 环境安装

```bash
# 1. 安装 Node.js (推荐使用 nvm)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 20
nvm use 20

# 2. 验证安装
node -v  # 应显示 v20.x.x
npm -v   # 应显示 10.x.x

# 3. 安装全局工具
npm install -g pnpm  # 推荐使用 pnpm
npm install -g @vue/cli  # Vue 项目
# 或
npm install -g create-react-app  # React 项目
```

### 1.3 项目克隆

```bash
# 1. 克隆项目
git clone https://github.com/your-org/TIANyu.git
cd TIANyu

# 2. 安装依赖
pnpm install
# 或
npm install

# 3. 启动开发服务器
pnpm dev
# 或
npm run dev

# 4. 访问 http://localhost:3000
```

### 1.4 环境变量配置

## 3. 开发工作流

### 3.1 标准开发流程

```
需求分析 → 设计评审 → 任务拆分 → 开发实现 → 代码审查 → 测试验证 → 部署上线
```

### 3.2 每日工作流

```bash
# 1. 开始工作前
git pull origin main              # 拉取最新代码
pnpm install                      # 安装依赖 (如有更新)

# 2. 创建功能分支
git checkout -b feature/xxx-feature

# 3. 开发过程中
pnpm dev                          # 启动开发服务器
# ... 编写代码 ...

# 4. 提交代码
git add .
git commit -m "feat: 添加 xxx 功能"

# 5. 推送分支
git push origin feature/xxx-feature

# 6. 创建 Pull Request
# 在 GitHub/GitLab 创建 PR，等待代码审查
```

### 3.3 Git 提交规范

```
<type>(<scope>): <subject>

<body>

<footer>
```

#### Type 类型

| 类型 | 说明 | 示例 |
|------|------|------|
| feat | 新功能 | `feat(button): 添加奶油米按钮` |
| fix | 修复 bug | `fix(input): 修复输入框聚焦样式` |
| docs | 文档更新 | `docs: 更新组件使用文档` |
| style | 代码格式 | `style: 格式化代码` |
| refactor | 重构 | `refactor: 重构表单验证逻辑` |
| test | 测试 | `test: 添加按钮组件测试` |
| chore | 构建/工具 | `chore: 更新依赖版本` |

#### 示例

```bash
# 功能开发
git commit -m "feat(button): 添加奶油米按钮样式

- 新增 btn-cream 类
- 添加悬停和点击效果
- 更新组件文档"

# Bug 修复
git commit -m "fix(input): 修复输入框在 iOS 上的聚焦问题

Closes #123"
```

### 4.4 Vue/React 规范

#### Vue 组件规范

```vue
<template>
  <div class="button" :class="classList" @click="handleClick">
    <slot></slot>
  </div>
</template>

<script>
export default {
  name: 'TButton',
  props: {
    type: {
      type: String,
      default: 'primary',
      validator: (value) => ['primary', 'secondary', 'cream'].includes(value)
    },
    size: {
      type: String,
      default: 'medium'
    },
    disabled: {
      type: Boolean,
      default: false
    }
  },
  computed: {
    classList() {
      return [`btn-${this.type}`, `btn-${this.size}`];
    }
  },
  methods: {
    handleClick(event) {
      if (this.disabled) return;
      this.$emit('click', event);
    }
  }
};
</script>

<style scoped>
.button {
  /* 样式 */
}
</style>
```

#### React 组件规范

```jsx
import React from 'react';
import PropTypes from 'prop-types';
import './Button.scss';

const Button = ({ 
  type = 'primary', 
  size = 'medium', 
  disabled = false,
  children,
  onClick 
}) => {
  const classList = `btn btn-${type} btn-${size}`;
  
  const handleClick = (event) => {
    if (disabled) return;
    onClick?.(event);
  };
  
  return (
    <button 
      className={classList} 
      disabled={disabled}
      onClick={handleClick}
    >
      {children}
    </button>
  );
};

Button.propTypes = {
  type: PropTypes.oneOf(['primary', 'secondary', 'cream']),
  size: PropTypes.oneOf(['small', 'medium', 'large']),
  disabled: PropTypes.bool,
  children: PropTypes.node.isRequired,
  onClick: PropTypes.func
};

export default Button;
```

### 6.4 API 对接

```javascript
// utils/request.js
import axios from 'axios';

const request = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL,
  timeout: 10000
});

// 请求拦截器
request.interceptors.request.use(
  config => {
    // 添加 token
    const token = localStorage.getItem('token');
    if (token) {
      config.headers.Authorization = `Bearer ${token}`;
    }
    return config;
  },
  error => {
    return Promise.reject(error);
  }
);

// 响应拦截器
request.interceptors.response.use(
  response => {
    return response.data;
  },
  error => {
    // 错误处理
    console.error('Request error:', error);
    return Promise.reject(error);
  }
);

export default request;

// pages/Product/index.vue
<script>
import request from '@/utils/request';

export default {
  data() {
    return {
      products: []
    };
  },
  async mounted() {
    await this.loadProducts();
  },
  methods: {
    async loadProducts() {
      try {
        this.loading = true;
        const data = await request.get('/products');
        this.products = data;
      } catch (error) {
        console.error('加载失败:', error);
      } finally {
        this.loading = false;
      }
    }
  }
};
</script>
```

### 7.2 单元测试示例

```javascript
// Button.test.js
import { describe, it, expect } from 'vitest';
import { mount } from '@vue/test-utils';
import Button from './Button.vue';

describe('Button', () => {
  it('renders default slot content', () => {
    const wrapper = mount(Button, {
      slots: { default: '点击我' }
    });
    expect(wrapper.text()).toBe('点击我');
  });
  
  it('applies correct type class', () => {
    const wrapper = mount(Button, {
      props: { type: 'primary' }
    });
    expect(wrapper.classes()).toContain('btn-primary');
  });
  
  it('emits click event', async () => {
    const wrapper = mount(Button);
    await wrapper.trigger('click');
    expect(wrapper.emitted('click')).toBeTruthy();
  });
  
  it('disables click when disabled', async () => {
    const wrapper = mount(Button, {
      props: { disabled: true }
    });
    await wrapper.trigger('click');
    expect(wrapper.emitted('click')).toBeFalsy();
  });
});
```

## 9. 版本管理

### 9.1 语义化版本

```
主版本号。次版本号。修订号
MAJOR.MINOR.PATCH
```

| 类型 | 说明 | 示例 |
|------|------|------|
| MAJOR | 不兼容的 API 变更 | 1.0.0 → 2.0.0 |
| MINOR | 向后兼容的功能 | 1.0.0 → 1.1.0 |
| PATCH | 向后兼容的 bug 修复 | 1.0.0 → 1.0.1 |

### 9.2 版本发布流程

```bash
# 1. 更新版本号
npm version patch  # 或 minor, major
```
