## 1. 项目介绍

### 1.1 什么是 TIANyu？

TIANyu 是一款 **AI 获客系统**的 WEB 端应用，旨在帮助企业主和运营人员通过 AI 工具提升获客效率。

### 1.2 核心特性

| 特性 | 说明 |
|------|------|
| 🎯 **AI 智能体广场** | 汇聚 18+ AI 工具，满足不同业务场景 |
| 🎨 **温暖设计** | 琥珀橙 + 奶油米配色，专业且亲切 |
| 📱 **响应式设计** | 完美适配移动端、平板、桌面端 |
| ⚡ **快速构建** | 基于 Next.js 14 + TypeScript + Tailwind CSS |

### 1.3 技术栈

```
TIANyu
├── 框架       Next.js 14 (App Router)
├── 语言       TypeScript
├── 样式       Tailwind CSS
├── 部署       Vercel
└── 包管理     pnpm
```

## 2. 环境准备

### 2.1 系统要求

| 软件 | 最低版本 | 推荐版本 |
|------|----------|----------|
| Node.js | v16.x | v20.x LTS |
| pnpm | v8.x | v9.x |
| Git | v2.x | v2.40+ |
| 浏览器 | Chrome 90+ | Chrome 120+ |

### 2.2 安装 Node.js

```bash
# macOS / Linux - 使用 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 20
nvm use 20

# Windows - 直接下载
# https://nodejs.org/
```

### 2.3 安装 pnpm

```bash
# 推荐：使用 corepack（Node.js 16+ 内置）
corepack enable
corepack prepare pnpm@latest --activate

# 或使用 npm
npm install -g pnpm
```

### 2.4 验证安装

```bash
node -v    # v20.x.x
pnpm -v    # 9.x.x
git -v     # 2.x.x
```

## 3. 快速启动

### 3.1 克隆项目

```bash
# 克隆仓库
git clone https://github.com/your-org/TIANyu.git

# 进入项目目录
cd TIANyu
```

### 3.2 安装依赖

```bash
# 使用 pnpm（推荐）
pnpm install

# 或使用 npm
npm install
```

### 3.3 配置环境变量

```bash
# 复制环境变量模板
cp .env.example .env.local

# 编辑环境变量
# .env.local 内容示例：
VITE_API_BASE_URL=https://api.tianyu.com
VITE_APP_TITLE=TIANyu
VITE_APP_VERSION=1.0.0
```

### 3.4 启动开发服务器

```bash
# 启动开发服务器
pnpm dev

# 输出：
# ▲ Next.js 14.x.x
# - Local:    http://localhost:3000
# - Network:  http://192.168.1.x:3000
```

### 3.5 访问项目

打开浏览器访问 **http://localhost:3000**，你将看到 TIANyu 主页。

│   │   └── ai-live-script.tsx # AI 直播脚本
│   ├── app/                    # App Router 页面
│   │   ├── layout.tsx          # 根布局
│   │   ├── page.tsx            # 首页

| 文件/目录 | 说明 |
|-----------|------|

## 5. 第一个页面

### 5.1 创建新页面

在 `src/app/` 下创建新页面：

```tsx
// src/app/hello/page.tsx
export default function HelloPage() {
  return (
    <div className="min-h-screen bg-[#FAF3E0] flex items-center justify-center">
      <div className="text-center">
        <h1 className="text-4xl font-bold text-[#1A1A1A] mb-4">
          欢迎来到 TIANyu
        </h1>
        <p className="text-lg text-[#666666]">
          你的 AI 获客助手
        </p>
      </div>
    </div>
  );
}
```

### 5.2 使用组件

```tsx
// 导入并使用组件
import Button from '@/components/Button';
import Card from '@/components/Card';
import Input from '@/components/Input';

export default function MyPage() {
  return (
    <Card>
      <Input placeholder="请输入内容" />
      <Button type="primary">提交</Button>
    </Card>
  );
}
```

### 5.3 添加样式

使用 Tailwind CSS 类名：

```tsx
// 快速样式
<div className="p-4 bg-white rounded-2xl shadow-sm">
  <h2 className="text-xl font-semibold text-gray-800">
    标题文字
  </h2>
</div>

// 使用设计令牌
<div className="p-4 bg-[var(--color-background)] rounded-[var(--radius-card)]">
  使用 CSS 变量
</div>
```

## 6. 常见问题

### Q1: pnpm install 失败怎么办？

```bash
# 清除缓存后重试
rm -rf node_modules
pnpm store prune
pnpm install
```

### Q2: 端口被占用？

```bash
# 指定端口启动
pnpm dev -- -p 3001

# 或修改 next.config.js
module.exports = {
  devServer: {
    port: 3001,
  },
}
```

### Q3: TypeScript 类型错误？

```bash
# 生成类型定义
pnpm dev &
pnpm tsc --init

# 或重启 TypeScript 服务器
# VS Code: Cmd+Shift+P → "TypeScript: Restart TS Server"
```

### Q4: 如何提交代码？

```bash
# 1. 创建分支
git checkout -b feature/your-feature

# 2. 添加更改
git add .

# 3. 提交
git commit -m "feat: 添加新功能"

# 4. 推送到远程
git push origin feature/your-feature
```
