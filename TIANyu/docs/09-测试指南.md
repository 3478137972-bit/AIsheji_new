## 1. 概述

### 1.1 测试目标

- 确保代码质量，减少 Bug
- 保障功能稳定性
- 支持持续集成/持续部署

### 1.2 测试金字塔

```
           /\
          /E2E\
         /------\
        /Integration\
       /------------\
      /  Unit Tests  \
     /________________\
     
- 单元测试：70% - 基础功能，快速反馈
- 集成测试：20% - 模块交互，流程验证
- E2E 测试：10% - 关键路径，端到端验证
```

### 1.3 技术栈

| 类型 | 工具 | 版本 |
|------|------|------|
| 测试框架 | Vitest | 1.x |
| 断言库 | Chai / expect | 4.x |
| 组件测试 | React Testing Library | 14.x |
| E2E 测试 | Playwright | 1.x |
| Mock | MSW | 2.x |
| 覆盖率 | Vitest Coverage | - |

## 2. 测试策略

### 2.1 测试原则

| 原则 | 说明 |
|------|------|
| **快速** | 单个测试 < 100ms |
| **独立** | 测试间无依赖，可并行 |
| **可重复** | 每次运行结果一致 |
| **自验证** | 断言明确，无需人工判断 |
| **及时** | 与代码同步编写 |

### 2.2 测试优先级

```
P0 - 核心功能
├── 用户登录/注册
├── 核心业务流程
└── 支付功能

P1 - 重要功能
├── 表单提交
├── 数据展示
└── 列表排序/筛选

P2 - 辅助功能
├── 样式变化
├── 动画效果
└── 边界处理
```

### 2.3 何时运行测试

- 本地开发：保存时自动运行
- 提交代码：pre-commit hook
- 合并代码：CI pipeline
- 发布前：完整测试套件

```typescript
// src/utils/format.test.ts
import { describe, it, expect } from 'vitest';
import { formatPrice, formatDate, truncateText } from './format';

describe('formatPrice', () => {
  it('应该正确格式化人民币', () => {
    expect(formatPrice(100)).toBe('¥100.00');
    expect(formatPrice(1234.5)).toBe('¥1,234.50');
  });

  it('应该处理零值', () => {
    expect(formatPrice(0)).toBe('¥0.00');
  });

  it('应该处理负数', () => {
    expect(formatPrice(-50)).toBe('¥-50.00');
  });
});

describe('truncateText', () => {
  it('应该截断超长文本', () => {
    expect(truncateText('Hello World', 5)).toBe('Hello...');
  });

  it('不应该截断短文本', () => {
    expect(truncateText('Hi', 5)).toBe('Hi');
  });
});
```

### 3.3 组件测试

```tsx
// src/components/Button/Button.test.tsx
import { render, screen, fireEvent } from '@testing-library/react';
import { describe, it, expect, vi } from 'vitest';
import Button from './Button';

describe('Button', () => {
  it('应该正确渲染文本', () => {
    render(<Button>点击我</Button>);
    expect(screen.getByText('点击我')).toBeInTheDocument();
  });

  it('点击时应该触发回调', () => {
    const handleClick = vi.fn();
    render(<Button onClick={handleClick}>点击我</Button>);
    
    fireEvent.click(screen.getByText('点击我'));
    expect(handleClick).toHaveBeenCalledTimes(1);
  });

  it('加载状态时应该禁用', () => {
    render(<Button loading>加载中</Button>);
    expect(screen.getByText('加载中')).toBeDisabled();
  });

  it('不同类型应该有不同的样式类', () => {
    const { rerender } = render(<Button type="primary">Primary</Button>);
    expect(screen.getByText('Primary')).toHaveClass('btn-primary');
    
    rerender(<Button type="secondary">Secondary</Button>);
    expect(screen.getByText('Secondary')).toHaveClass('btn-secondary');
  });
});
```

### 3.4 Hook 测试

```tsx
// src/hooks/useCounter.test.ts
import { renderHook, act } from '@testing-library/react';
import { describe, it, expect } from 'vitest';
import { useCounter } from './useCounter';

describe('useCounter', () => {
  it('应该初始化为默认值', () => {
    const { result } = renderHook(() => useCounter());
    expect(result.current.count).toBe(0);
  });

  it('应该支持初始值', () => {
    const { result } = renderHook(() => useCounter(10));
    expect(result.current.count).toBe(10);
  });

  it('increment 应该增加计数', () => {
    const { result } = renderHook(() => useCounter());
    
    act(() => {
      result.current.increment();
    });
    
    expect(result.current.count).toBe(1);
  });

  it('decrement 应该减少计数', () => {
    const { result } = renderHook(() => useCounter(5));
    
    act(() => {
      result.current.decrement();
    });
    
    expect(result.current.count).toBe(4);
  });

  it('reset 应该重置为初始值', () => {
    const { result } = renderHook(() => useCounter(5));
    
    act(() => {
      result.current.increment();
      result.current.increment();
      result.current.reset();
    });
    
    expect(result.current.count).toBe(5);
  });
});
```

### 4.1 API 集成测试

使用 MSW (Mock Service Worker) 模拟 API：

```typescript
// src/mocks/handlers.ts
import { http, HttpResponse } from 'msw';

export const handlers = [
  // 模拟获取用户信息
  http.get('/api/v1/user/info', () => {
    return HttpResponse.json({
      code: 0,
      message: 'success',
      data: {
        id: '1',
        name: '张三',
        email: 'zhangsan@example.com',
      },
    });
  }),

  // 模拟生成直播脚本
  http.post('/api/v1/ai/live-script', async ({ request }) => {
    const body = await request.json() as any;
    
    return HttpResponse.json({
      code: 0,
      message: 'success',
      data: {
        开场话术: `欢迎来到直播间！今天给大家介绍${body.productName}...`,
        产品介绍: body.productFeatures,
        促单话术: '原价199，现在只要99！',
        互动引导: '喜欢的宝宝们关注一下',
        结束话术: '感谢大家的观看，明天见！',
      },
    });
  }),
];
```

### 4.2 页面集成测试

```tsx
// src/pages/__tests__/live-script.test.tsx
import { render, screen, waitFor, fireEvent } from '@testing-library/react';
import { setupServer } from 'msw/node';
import { handlers } from '@/mocks/handlers';
import LiveScriptPage from '@/pages/live-script';

const server = setupServer(...handlers);

beforeAll(() => server.listen());
afterEach(() => server.resetHandlers());
afterAll(() => server.close());

describe('LiveScriptPage', () => {
  it('应该正确提交表单并显示结果', async () => {
    render(<LiveScriptPage />);
    
    // 填写表单
    fireEvent.change(screen.getByLabelText('产品名称'), {
      target: { value: '测试产品' },
    });
    
    fireEvent.change(screen.getByLabelText('产品卖点'), {
      target: { value: '高品质、低价格' },
    });
    
    // 提交
    fireEvent.click(screen.getByText('生成脚本'));
    
    // 等待结果
    await waitFor(() => {
      expect(screen.getByText(/欢迎来到直播间/)).toBeInTheDocument();
    });
  });
});
```

### 5.1 Playwright 配置

```typescript
// playwright.config.ts
import { defineConfig, devices } from '@playwright/test';

export default defineConfig({
  testDir: './e2e',
  fullyParallel: true,
  forbidOnly: !!process.env.CI,
  retries: process.env.CI ? 2 : 0,
  workers: process.env.CI ? 1 : undefined,

    url: 'http://localhost:3000',

  },
});
```

### 5.2 E2E 测试示例

```typescript
// e2e/live-script.spec.ts
import { test, expect } from '@playwright/test';

test.describe('AI 直播脚本', () => {
  test('完整流程：填写表单 → 生成脚本', async ({ page }) => {
    // 1. 访问页面
    await page.goto('/ai-live-script');
    
    // 2. 验证页面标题
    await expect(page.locator('h1')).toContainText('AI 直播脚本');
    
    // 3. 填写表单
    await page.fill('input[name="productName"]', '测试产品');
    await page.fill('textarea[name="productFeatures"]', '高品质、低价格');
    await page.selectOption('select[name="targetAudience"]', '年轻人');
    await page.selectOption('select[name="duration"]', '30min');
    
    // 4. 点击生成按钮
    await page.click('button:text("生成脚本")');
    
    // 5. 等待加载完成
    await page.waitForSelector('.loading', { state: 'hidden' });
    
    // 6. 验证结果
    await expect(page.locator('text=开场话术')).toBeVisible();
    await expect(page.locator('text=欢迎来到直播间')).toBeVisible();
  });

  test('表单验证：必填字段', async ({ page }) => {
    await page.goto('/ai-live-script');
    
    // 直接点击提交
    await page.click('button:text("生成脚本")');
    
    // 验证错误提示
    await expect(page.locator('text=请输入产品名称')).toBeVisible();
  });

  test('响应式：移动端布局', async ({ page }) => {
    // 设置移动端视口
    await page.setViewportSize({ width: 375, height: 667 });
    
    await page.goto('/ai-live-script');
    
    // 验证移动端特有的元素
    await expect(page.locator('.mobile-tab-bar')).toBeVisible();
  });
});
```

## 7. CI/CD 集成

### 7.1 GitHub Actions

```yaml
# .github/workflows/test.yml
name: Test

on:
  push:
    branches: [main, develop]
  pull_request:
    branches: [main]

jobs:
  test:
    runs-on: ubuntu-latest
    
    steps:
      - uses: actions/checkout@v4
      
      - uses: pnpm/action-setup@v2
        with:
          version: 9
      
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: 'pnpm'
      
      - name: Install dependencies
        run: pnpm install
      
      - name: Run type check
        run: pnpm type-check
      
      - name: Run unit tests
        run: pnpm test:run
      
      - name: Run E2E tests
        run: pnpm test:e2e
      
      - name: Upload coverage
        uses: codecov/codecov-action@v3
        with:
          files: ./coverage/coverage-final.json
```

### 7.2 本地 pre-commit

```bash
# 安装 husky
pnpm add -D husky

# 初始化
npx husky init

# 添加 pre-commit 脚本
echo "pnpm test:run" > .husky/pre-commit

# 安装 lint-staged
pnpm add -D lint-staged
```
