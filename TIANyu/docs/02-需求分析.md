## 1. 项目概述

### 1.1 项目背景

TIANyu 是一款面向企业主的 **AI 获客系统**，旨在通过 AI 技术帮助用户提升获客效率、降低运营成本。项目从小程序端起步，现已扩展至 WEB 端，为用户提供更丰富的交互体验。

### 1.2 项目目标

| 目标 | 指标 | 优先级 |
|------|------|--------|
| 提供 18+ AI 工具 | 覆盖主流获客场景 | P0 |
| 优质用户体验 | 页面加载 < 2s | P0 |
| 跨平台兼容 | 支持 3 种设备类型 | P1 |
| 快速迭代 | 周级发布新功能 | P1 |

### 1.3 核心价值

```
用户痛点                        解决方案
├─ 获客成本高              →    AI 智能推荐，降低试错成本
├─ 内容创作效率低          →    AI 辅助生成，提升 10 倍效率
├─ 运营策略不清晰          →    数据驱动决策，精准触达
└─ 多平台管理复杂          →    一站式管理，统一入口
```

## 2. 用户画像

### 2.1 核心用户

| 用户类型 | 典型特征 | 核心需求 |
|----------|----------|----------|
| **企业主/老板** | 30-45 岁，营收千万级 | 快速获取流量，提升业绩 |
| **私域运营** | 25-35 岁，运营岗位 | 高效产出内容，管理客户 |
| **自媒体博主** | 22-35 岁，内容创作者 | 创作爆款，提升影响力 |
| **电商商家** | 25-40 岁，电商从业者 | 引流获客，提升转化 |

### 2.2 用户旅程

```
┌─────────────────────────────────────────────────────────┐
│                    用户旅程地图                          │
├─────────────────────────────────────────────────────────┤
│  认知 → 兴趣 → 尝试 → 使用 → 复购 → 推荐                │
│  │     │     │    │     │     │                         │
│  ▼     ▼     ▼    ▼     ▼     ▼                         │
│  landing  功能页  免费用  会员    续费    分享           │
│  页面     介绍    户     付费    付费    传播           │
└─────────────────────────────────────────────────────────┘
```

### 2.3 用户场景

| 场景 | 描述 | 痛点 |
|------|------|------|
| **直播脚本** | 电商直播话术撰写 | 写稿耗时，缺少创意 |
| **短视频选题** | 抖音/视频号内容规划 | 选题困难，数据分析难 |
| **私域运营** | 社群内容规划 | 内容匮乏，互动性差 |
| **账号分析** | 竞品账号拆解 | 数据分散，分析耗时 |

### 3.3 功能详情

#### F04 AI 直播脚本（已完成）

**功能描述**：根据输入的产品信息、目标受众、直播时长等参数，AI 自动生成直播话术脚本。

**表单字段**：
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 产品名称 | 文本 | 是 | 产品名称 |
| 产品卖点 | 多行文本 | 是 | 产品核心卖点 |
| 目标受众 | 选择 | 是 | 人群画像 |
| 直播时长 | 选择 | 是 | 15min/30min/60min/90min |
| 直播风格 | 多选 | 否 | 活泼/专业/亲和... |

**输出内容**：
- 开场话术
- 产品介绍
- 促单话术
- 互动引导
- 结束话术

## 4. 非功能需求

### 4.1 性能需求

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 首屏加载 | < 2s | 首字节到可交互 |
| 接口响应 | < 500ms | API 平均响应时间 |
| 交互延迟 | < 100ms | 用户操作反馈 |
| FPS | ≥ 30 | 页面流畅度 |

### 4.2 兼容性需求

| 平台 | 最低版本 | 说明 |
|------|----------|------|
| Chrome | 90+ | 推荐最新 |
| Safari | 14+ | iOS Safari |
| Edge | 90+ | Chromium Edge |
| Firefox | 88+ | 推荐最新 |

### 4.3 响应式断点

| 设备类型 | 断点 | 网格列数 |
|----------|------|----------|
| 移动端 | < 768px | 2 列 |
| 平板 | 768px - 1024px | 3 列 |
| 桌面 | > 1024px | 4 列 |

### 4.4 可用性需求

- [ ] 支持键盘导航
- [ ] 颜色对比度 WCAG AA
- [ ] 表单字段必填提示
- [ ] 加载状态反馈
- [ ] 错误状态提示
