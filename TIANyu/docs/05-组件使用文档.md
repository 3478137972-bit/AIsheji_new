## 1. 按钮 (Button)

### 1.1 基础用法

#### 主按钮 (Primary)

```html
<!-- HTML -->
<button class="btn btn-primary">
  主按钮
</button>

<!-- Vue -->
<template>
  <t-button type="primary">主按钮</t-button>
</template>

<!-- React -->
<Button type="primary">主按钮</Button>
```

```css
/* CSS */
.btn-primary {
  height: 48px;
  padding: 0 32px;
  background: linear-gradient(135deg, #E67E22 0%, #D35400 100%);
  border-radius: 24px;
  color: #FFFFFF;
  font-size: 16px;
  font-weight: 600;
  border: none;
  box-shadow: 0 2px 12px rgba(230, 126, 34, 0.3);
  cursor: pointer;
  transition: all 0.2s ease-out;
}

.btn-primary:hover {
  box-shadow: 0 4px 16px rgba(230, 126, 34, 0.4);
  transform: translateY(-1px);
}

.btn-primary:active {
  transform: translateY(0);
  box-shadow: 0 2px 8px rgba(230, 126, 34, 0.3);
}

.btn-primary:disabled {
  background: #E0E0E0;
  color: #999999;
  box-shadow: none;
  cursor: not-allowed;
}
```

#### 次按钮 (Secondary)

```html
<button class="btn btn-secondary">
  次按钮
</button>
```

```css
.btn-secondary {
  height: 48px;
  padding: 0 32px;
  background: #FFFFFF;
  border-radius: 24px;
  color: #E67E22;
  font-size: 16px;
  font-weight: 600;
  border: 1px solid #E67E22;
  cursor: pointer;
  transition: all 0.2s ease-out;
}

.btn-secondary:hover {
  background: #FAF3E0;
  border-color: #D35400;
}

.btn-secondary:active {
  background: #F5E6D3;
}
```

#### 奶油米按钮 (Cream)

```html
<button class="btn btn-cream">
  奶油米按钮
</button>
```

```css
.btn-cream {
  height: 48px;
  padding: 0 32px;
  background: #FAF3E0;
  border-radius: 24px;
  color: #E67E22;
  font-size: 16px;
  font-weight: 600;
  border: none;
  cursor: pointer;
  transition: all 0.2s ease-out;
}

.btn-cream:hover {
  background: #F5E6D3;
}
```


### 1.3 按钮状态



## 2. 输入框 (Input)

### 2.1 基础用法

#### 单行输入

```html
<div class="form-field">
  <input 
    type="text" 
    class="input-single" 
    placeholder="请输入内容"
  />
</div>
```

```css
.input-single {
  height: 48px;
  padding: 12px 16px;
  border: 1px solid #E0E0E0;
  border-radius: 8px;
  font-size: 16px;
  color: #333333;
  background: #FFFFFF;
  transition: all 0.2s ease-out;
}

.input-single:focus {
  border-color: #E67E22;
  outline: none;
  box-shadow: 0 0 0 2px rgba(230, 126, 34, 0.1);
}

.input-single:hover:not(:focus) {
  border-color: #D35400;
}

.input-single::placeholder {
  color: #999999;
  font-size: 14px;
}
```

#### 多行输入

```html
<div class="form-field">
  <textarea 
    class="input-multiple"
    placeholder="请输入多行内容"
    rows="4"
  ></textarea>
</div>
```

```css
.input-multiple {
  min-height: 80px;
  padding: 12px 16px;
  border: 1px solid #E5E5E5;
  border-radius: 8px;
  font-size: 16px;
  color: #333333;
  background: #FFFFFF;
  resize: vertical;
  font-family: inherit;
}
```

### 2.2 带标签的输入框

```html
<div class="form-field">
  <label class="field-label">
    <span class="field-number">1.</span>

    <span class="field-required">*</span>
  </label>

    type="text" 
    class="input-single" 
    placeholder="请输入标题"
    required
  />
  <span class="field-hint">标题将用于搜索和分类</span>
</div>
```

```css
.form-field {
  margin-bottom: 20px;
}

.field-label {
  display: block;

  margin-left: 8px;
}

.field-hint {
  display: block;
  margin-top: 8px;
  font-size: 12px;
  color: #999999;
}
```

### 2.3 输入框状态

```html
<!-- 默认状态 -->
<input class="input-single" type="text" />

<!-- 聚焦状态 -->
<input class="input-single" type="text" autofocus />

<!-- 错误状态 -->
<input class="input-single error" type="text" value="无效内容" />
<span class="error-message">请输入有效的邮箱地址</span>

<!-- 成功状态 -->
<input class="input-single success" type="text" value="正确内容" />

<!-- 禁用状态 -->
<input class="input-single" type="text" disabled value="不可编辑" />

<!-- 只读状态 -->
<input class="input-single" type="text" readonly value="只读内容" />
```

```css
.input-single.error {
  border-color: #FF4D4F;
  background: rgba(255, 77, 79, 0.05);
}

.input-single.success {
  border-color: #52C41A;
}

.input-single:disabled,
.input-single:read-only {
  background: #F0F0F0;
  color: #999999;
  cursor: not-allowed;
}
```

### 2.4 输入框类型

```html
<!-- 文本输入 -->
<input class="input-single" type="text" placeholder="文本" />

<!-- 邮箱输入 -->
<input class="input-single" type="email" placeholder="邮箱" />

<!-- 数字输入 -->
<input class="input-single" type="number" placeholder="数字" />

<!-- 密码输入 -->
<input class="input-single" type="password" placeholder="密码" />

<!-- 电话输入 -->
<input class="input-single" type="tel" placeholder="手机号" />

<!-- URL 输入 -->
<input class="input-single" type="url" placeholder="网址" />

<!-- 日期输入 -->
<input class="input-single" type="date" />

<!-- 时间输入 -->
<input class="input-single" type="time" />
```

### 2.5 带前缀/后缀的输入框

```html
<!-- 前缀图标 -->
<div class="input-wrapper">
  <span class="input-prefix">
    <span class="icon icon-search"></span>
  </span>
  <input class="input-single" type="text" placeholder="搜索..." />
</div>

<!-- 后缀按钮 -->
<div class="input-wrapper">
  <input class="input-single" type="text" placeholder="输入网址" />
  <span class="input-suffix">
    <button class="btn-icon">
      <span class="icon icon-link"></span>
    </button>
  </span>
</div>

<!-- 前后缀都有 -->
<div class="input-wrapper">
  <span class="input-prefix">¥</span>
  <input class="input-single" type="number" placeholder="0.00" />
  <span class="input-suffix">元</span>
</div>
```

```css
.input-wrapper {
  position: relative;
  display: flex;
  align-items: center;
}

.input-prefix,
.input-suffix {
  position: absolute;
  color: #999999;
  font-size: 14px;
  pointer-events: none;
}

.input-prefix {
  left: 16px;
}

.input-suffix {
  right: 16px;
}

.input-wrapper .input-single {
  width: 100%;
  padding-left: 40px;  /* 有前缀时 */
  padding-right: 40px; /* 有后缀时 */
}
```
