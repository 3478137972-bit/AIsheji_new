## 1. 响应式设计原则

### 1.1 移动优先 (Mobile First)

**核心理念：从小屏到大屏渐进增强**

```css
/* ✅ 正确：移动优先 */
/* 基础样式 - 移动端 */
.container {
  padding: 0 16px;
}

/* 平板 */
@media (min-width: 768px) {
  .container {
    padding: 0 24px;
  }
}

/* 桌面 */
@media (min-width: 1024px) {
  .container {
    padding: 0 32px;
    max-width: 1200px;
    margin: 0 auto;
  }
}

/* ❌ 错误：桌面优先 */
.container {
  max-width: 1200px;
  margin: 0 auto;
}

@media (max-width: 1023px) {
  .container {
    max-width: 960px;
  }
}

@media (max-width: 767px) {
  .container {
    padding: 0 16px;
  }
}
```

### 1.2 流体布局

**使用相对单位而非固定单位**

```css
/* ✅ 推荐：使用百分比、fr、vw */
.grid {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  gap: 4%; /* 相对间距 */
}

.card {
  width: 100%; /* 自适应宽度 */
  max-width: 400px; /* 最大宽度限制 */
}

/* ❌ 避免：固定像素宽度 */
.grid {
  width: 750px; /* 固定宽度 */
}

.card {
  width: 375px; /* 固定宽度 */
}
```

### 1.3 内容优先

**根据内容决定断点，而非设备**

```css
/* ✅ 根据内容需求设置断点 */
.card-grid {
  display: grid;
  gap: 16px;
}

/* 当卡片宽度小于 280px 时换行 */
@media (min-width: 600px) {
  .card-grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (min-width: 920px) {
  .card-grid {
    grid-template-columns: repeat(3, 1fr);
  }
}

@media (min-width: 1240px) {
  .card-grid {
    grid-template-columns: repeat(4, 1fr);
  }
}
```

### 1.4 渐进增强

**基础功能全平台，高级功能按需加载**

```css
/* 基础样式 - 所有设备 */
.button {
  padding: 12px 24px;
  background: #E67E22;
  color: #FFFFFF;
  border-radius: 24px;
}

/* 增强：悬停效果 - 仅鼠标设备 */
@media (hover: hover) {
  .button:hover {
    background: #D35400;
    transform: translateY(-2px);
  }
}

/* 增强：焦点效果 - 键盘导航 */
.button:focus-visible {
  outline: 2px solid #E67E22;
  outline-offset: 2px;
}
```

## 2. 断点系统

### 2.1 标准断点

```css
:root {
  /* TIANyu 响应式断点 */
  --breakpoint-xs: 375px;   /* 小屏手机 */
  --breakpoint-sm: 640px;   /* 大屏手机/小平板 */
  --breakpoint-md: 768px;   /* 平板 (竖屏) */
  --breakpoint-lg: 1024px;  /* 平板 (横屏)/小屏电脑 */
  --breakpoint-xl: 1280px;  /* 标准电脑 */
  --breakpoint-2xl: 1536px; /* 大屏电脑 */
}
```

### 2.2 断点使用场景

| 断点 | 最小宽度 | 典型设备 | 布局调整 |
|------|----------|----------|----------|
| xs | 375px | iPhone SE, 小屏手机 | 基础布局 |
| sm | 640px | iPhone Plus, 大屏手机 | 微调间距 |
| md | 768px | iPad, 平板 (竖屏) | 2 列→3 列，底部 Tab→顶部导航 |
| lg | 1024px | iPad Pro, 平板 (横屏) | 3 列→4 列，显示侧边栏 |
| xl | 1280px | 笔记本，小屏电脑 | 最大宽度限制 |
| 2xl | 1536px | 台式机，大屏显示器 | 更大间距 |

### 2.3 媒体查询 Mixins

```scss
// SCSS Mixins
@mixin xs {
  @media (min-width: 375px) { @content; }
}

@mixin sm {
  @media (min-width: 640px) { @content; }
}

@mixin md {
  @media (min-width: 768px) { @content; }
}

@mixin lg {
  @media (min-width: 1024px) { @content; }
}

@mixin xl {
  @media (min-width: 1280px) { @content; }
}

@mixin 2xl {
  @media (min-width: 1536px) { @content; }
}

// 使用示例
.container {
  padding: 0 16px;
  
  @include md {
    padding: 0 24px;
  }
  
  @include lg {
    padding: 0 32px;
    max-width: 1200px;
  }
  
  @include 2xl {
    max-width: 1440px;
  }
}
```

### 2.4 CSS 变量断点

```css
/* 在 JavaScript 中使用 */
const breakpoints = {
  xs: 375,
  sm: 640,
  md: 768,
  lg: 1024,
  xl: 1280,
  '2xl': 1536
};

// 监听窗口大小变化
function handleResize() {
  const width = window.innerWidth;
  
  if (width >= breakpoints['2xl']) {
    document.body.classList.add('viewport-2xl');
  } else {
    document.body.classList.remove('viewport-2xl');
  }
  
  // ... 其他断点
}

window.addEventListener('resize', handleResize);
```

### 5.4 图片懒加载

```html
<img 
  src="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7"
  data-src="/images/photo.jpg"
  alt="描述文字"
  class="lazyload"
/>
```

```javascript
// 使用 Intersection Observer 实现懒加载
const lazyImages = document.querySelectorAll('img.lazyload');

const imageObserver = new IntersectionObserver((entries, observer) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      const img = entry.target;
      img.src = img.dataset.src;
      img.classList.remove('lazyload');
      observer.unobserve(img);
    }
  });
});

lazyImages.forEach(img => imageObserver.observe(img));
```

```javascript
// 根据设备像素比加载合适图片
function getOptimalImage(baseUrl) {
  const dpr = window.devicePixelRatio || 1;
  const width = window.innerWidth;
  
  let suffix = '@1x';
  if (dpr >= 2 && width >= 768) {
    suffix = '@2x';
  }
  if (dpr >= 3 && width >= 1024) {
    suffix = '@3x';
  }
  
  return `${baseUrl}${suffix}.jpg`;
}
```

```javascript
// 批量 DOM 操作
function updateMultipleElements() {
  const fragment = document.createDocumentFragment();
  
  for (let i = 0; i < 100; i++) {
    const div = document.createElement('div');
    div.textContent = `Item ${i}`;
    fragment.appendChild(div);
  }
  
  document.body.appendChild(fragment); // 单次重排
}
```
