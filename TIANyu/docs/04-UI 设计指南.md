## 1. 设计理念

### 1.1 核心愿景

**温暖活力 × 高级轻奢 × 现代简洁**

TIANyu 的 UI 设计旨在打造一个既专业又亲切的 AI 工具平台，让用户在高效完成任务的同时，感受到温暖舒适的使用体验。

### 1.2 设计关键词

| 关键词 | 含义 | 体现方式 |
|--------|------|----------|
| **温暖** | 亲和力、舒适感 | 琥珀橙主色调、奶油米背景、圆角设计 |
| **活力** | 行动力、创造力 | 渐变效果、动效反馈、鲜明对比 |
| **高级** | 品质感、精致感 | 细腻阴影、恰当留白、精致细节 |
| **简洁** | 清晰、高效 | 明确层次、减少干扰、直观操作 |

### 1.3 设计目标

```
用户目标                          设计目标
├─ 快速找到所需 AI 工具    →    清晰的导航与分类
├─ 轻松完成表单填写        →    友好的输入体验
├─ 获得专业级输出          →    精致的视觉呈现
└─ 享受使用过程            →    温暖的交互反馈
```

## 2. 设计原则

### 2.1 一致性原则

**全站统一的设计语言**

- 使用统一的色彩系统
- 遵循相同的间距规范
- 保持组件样式一致
- 统一的交互反馈

```css
/* ✅ 正确：使用 CSS 变量保持一致性 */
.button {
  background: var(--color-primary);
  padding: var(--spacing-button-padding);
  border-radius: var(--radius-button);
}

/* ❌ 错误：硬编码数值 */
.button {
  background: #E67E22;
  padding: 12px 24px;
  border-radius: 24px;
}
```

### 2.2 层次原则

**清晰的信息层级**

```
视觉层次（从重要到次要）：
1. 页面标题 (24px Bold, #1A1A1A)
2. 模块标题 (20px SemiBold, #333333)
3. 卡片标题 (18px SemiBold, #333333)
4. 正文内容 (16px Regular, #666666)
5. 辅助信息 (14px Regular, #999999)
```

### 2.3 对比原则

**通过对比突出重点**

| 对比类型 | 应用场景 | 示例 |
|----------|----------|------|
| 色彩对比 | 主按钮 vs 次按钮 | 琥珀橙渐变 vs 白色描边 |
| 大小对比 | 标题 vs 正文 | 24px vs 16px |
| 粗细对比 | 重点文字 vs 普通文字 | Bold vs Regular |
| 空间对比 | 内容区 vs 留白 | 卡片内边距 16px |

### 2.4 反馈原则

**每个操作都有反馈**

```css
/* 按钮状态反馈 */
.btn-primary {
  /* 默认状态 */
  box-shadow: 0 2px 12px rgba(230, 126, 34, 0.3);
  transition: all 0.2s ease-out;
}

.btn-primary:hover {
  /* 悬停状态 - 阴影加深，轻微上移 */
  box-shadow: 0 4px 16px rgba(230, 126, 34, 0.4);
  transform: translateY(-1px);
}

.btn-primary:active {
  /* 点击状态 - 阴影还原，轻微缩放 */
  transform: scale(0.98);
  box-shadow: 0 2px 8px rgba(230, 126, 34, 0.3);
}
```

### 2.5 效率原则

**减少用户认知负担**

- 常用操作放在显眼位置
- 表单字段合理分组
- 提供清晰的引导文案
- 减少不必要的步骤

## 3. 色彩应用指南

### 3.1 主色调应用

#### 琥珀橙 (#E67E22) - 品牌色

**使用场景：**

| 元素 | 用法 | 占比 |
|------|------|------|
| 主按钮背景 | 渐变起点 | 60% |
| 激活状态 | 纯色 | 20% |
| 文字强调 | 文字颜色 | 10% |
| 图标 | 图标颜色 | 10% |

**示例：**

```css
/* 主按钮 - 渐变应用 */
.btn-primary {
  background: linear-gradient(135deg, #E67E22 0%, #D35400 100%);
}

/* 激活 Tab - 纯色应用 */
.tab-item.active {
  background: #E67E22;
  color: #FFFFFF;
}

/* 链接文字 - 强调应用 */
.link-amber {
  color: #E67E22;
  text-decoration: underline;
}
```

### 3.2 辅助色应用

#### 奶油米 (#FAF3E0) - 背景装饰

**使用场景：**

```css
/* 页面背景渐变 */
.page-background {
  background: linear-gradient(180deg, #FAF3E0 0%, #FFFFFF 100%);
}

/* 卡片装饰背景 */
.card-cream {
  background: linear-gradient(135deg, #FFFFFF 0%, #FAF3E0 100%);
}

/* 选中态背景 */
.selected-item {
  background: #FAF3E0;
}
```

#### 暖米色 (#F5E6D3) - 悬停态

```css
/* 卡片悬停背景 */
.card:hover {
  background: #F5E6D3;
}

/* Tab 悬停态 */
.tab-item:hover {
  background: rgba(230, 126, 34, 0.1);
}
```

### 3.3 中性色应用

#### 文字颜色层次

```css
/* 标题文字 - 最强对比 */
.page-title {
  color: #1A1A1A;
}

/* 正文文字 - 标准对比 */
.body-text {
  color: #333333;
}

/* 次要文字 - 较弱对比 */
.secondary-text {
  color: #666666;
}

/* 辅助文字 - 最弱对比 */
.hint-text {
  color: #999999;
}
```

### 3.4 功能色应用

#### 状态提示

```css
/* 成功状态 */
.success-message {
  color: #52C41A;
  background: rgba(82, 196, 26, 0.1);
}

/* 错误状态 */
.error-message {
  color: #FF4D4F;
  background: rgba(255, 77, 79, 0.1);
}

/* 警告状态 */
.warning-message {
  color: #F1C40F;
  background: rgba(241, 196, 15, 0.1);
}

/* 信息提示 */
.info-message {
  color: #3498DB;
  background: rgba(52, 152, 219, 0.1);
}
```

### 3.5 色彩搭配禁忌

| ❌ 避免 | ✅ 推荐 |
|--------|--------|
| 过多高饱和色彩 | 主色 + 中性色为主 |
| 纯黑 (#000000) | 标题黑 (#1A1A1A) |
| 纯灰 (#808080) | 冷暖灰搭配 |
| 超过 3 种主色 | 1 主色 + 2 辅助色 |

## 4. 字体与排版

### 4.1 字号使用场景

#### 标题层级

```
H1 - 页面标题 (24px Bold)
├─ 用于：页面主标题
├─ 颜色：#1A1A1A
└─ 位置：页面顶部，导航栏下方

H2 - 模块标题 (20px SemiBold)
├─ 用于：内容模块标题
├─ 颜色：#333333
└─ 位置：模块顶部

H3 - 卡片标题 (18px SemiBold)
├─ 用于：卡片/区块标题
├─ 颜色：#333333
└─ 位置：卡片内部顶部
```

#### 正文层级

```
Body Large (16px Regular)
├─ 用于：主要内容、按钮文字
├─ 颜色：#333333 / #666666
└─ 行高：1.6

Body (14px Regular)
├─ 用于：说明文字、表单标签
├─ 颜色：#666666
└─ 行高：1.5

Small (12px Regular)
├─ 用于：标签、提示、消耗说明
├─ 颜色：#999999
└─ 行高：1.4
```

### 4.2 字体排印最佳实践

#### 行高控制

```css
/* 标题行高 - 紧凑 */
h1, h2, h3 {
  line-height: 1.4;
}

/* 正文行高 - 舒适 */
p {
  line-height: 1.6;
}

/* 说明文字行高 - 适中 */
.small-text {
  line-height: 1.5;
}
```

#### 字间距控制

```css
/* 中文文本 - 无需额外字间距 */
.chinese-text {
  letter-spacing: 0;
}

/* 英文大写 - 适当增加字间距 */
.english-uppercase {
  letter-spacing: 0.5px;
}

/* 数字显示 - 使用等宽字体 */
.number-display {
  font-family: 'SF Mono', 'Monaco', monospace;
  font-variant-numeric: tabular-nums;
}
```

### 4.3 文本截断策略

```css
/* 单行截断 */
.text-truncate-1 {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;

/* 多行截断 (2 行) */
.text-truncate-2 {
  display: -webkit-box;
  -webkit-line-clamp: 2;
  -webkit-box-orient: vertical;
  overflow: hidden;

  display: -webkit-box;
```

## 6. 组件设计模式

### 6.1 按钮设计模式

#### 按钮层级

```
主按钮 (Primary)
├─ 用途：主要操作、确认提交
├─ 样式：琥珀橙渐变
└─ 位置：右侧或底部

次按钮 (Secondary)
├─ 用途：次要操作、取消返回
├─ 样式：白色背景 + 琥珀橙描边
└─ 位置：左侧或与主按钮并排

奶油米按钮 (Cream)
├─ 用途：特殊场景、温和提示
├─ 样式：奶油米背景
└─ 位置：根据场景

禁用按钮 (Disabled)
├─ 用途：不可用状态
├─ 样式：灰色背景
└─ 触发：表单未填写、条件不满足
```

#### 按钮尺寸

```css
/* 大按钮 - 主要操作 */
.btn-large {
  height: 48px;
  padding: 0 32px;
  font-size: 16px;
}

/* 中按钮 - 常规操作 */
.btn-medium {
  height: 40px;
  padding: 0 24px;
  font-size: 14px;
}

/* 小按钮 - 紧凑场景 */
.btn-small {
  height: 32px;
  padding: 0 16px;
  font-size: 12px;
}
```

## 8. 响应式设计策略

### 8.1 移动优先

```css
/* 基础样式 - 移动端 */
.container {
  padding: 0 16px;
}

/* 平板适配 */
@media (min-width: 768px) {
  .container {
    padding: 0 24px;
  }
}

/* 桌面适配 */
@media (min-width: 1024px) {
  .container {
    padding: 0 32px;
    max-width: 1200px;
    margin: 0 auto;
  }
}
```

### 8.2 断点使用指南

| 断点 | 最小宽度 | 目标设备 | 布局调整 |
|------|----------|----------|----------|
| sm | 640px | 小屏手机 (横屏) | 微调间距 |
| md | 768px | 平板 (竖屏) | 3 列网格、顶部导航 |
| lg | 1024px | 平板 (横屏) / 小屏电脑 | 4 列网格、侧边栏 |
| xl | 1280px | 标准电脑 | 最大宽度限制 |
| 2xl | 1536px | 大屏电脑 | 更大间距 |
