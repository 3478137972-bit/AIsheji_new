### 1. GitHub 仓库配置问题
- **正确仓库**: `https://github.com/3478137972-bit/AI_HuoKe.git`
- **错误仓库**: `miaodongAIchaojiyaungong_design` (设计项目)
- **问题根源**: 全局 Git 配置有 `insteadof` 规则自动替换 URL
- **解决方案**: 
  ```bash
  git config --global --unset "url.https://ghp_...@github.com/.insteadof"
  git config --global url."https://ghp_...@github.com/3478137972-bit/AI_HuoKe.git".insteadOf "https://github.com/3478137972-bit/AI_HuoKe.git"
  ```
